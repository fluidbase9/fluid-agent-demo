# fluid-agent-demo

> Full working demo of **Fluid Wallet API + FADP/1.0** for developers.

Clone it, run the CLI once, and see a real AI agent auto-pay for live crypto data.

## What's inside

| File | What it does |
|------|-------------|
| `server/index.js` | Express API with 7 FADP-gated endpoints — prices, markets, swap quotes, identity, balance |
| `agent/index.js` | Interactive CLI agent — prompts you to pay, auto-pays via Fluid Wallet, prints results |

## Quick start

```bash
# 1. Clone & install
git clone https://github.com/fluidbase9/fluid-agent-demo
cd fluid-agent-demo
npm install

# 2. Run the setup CLI (generates ALL keys automatically)
npx @fluidwallet/fadp-cli@latest
# ✓ Creates developer account
# ✓ Generates FLDP EC key pair → .env.fadp
# ✓ Generates FLUID_AGENT_KEY (fwag_...) → .env.fadp + ~/.zshrc
# ✓ Installs agent skills into Codex / Cursor / Cline / etc.
# ✓ Scaffolds fadp-sample/ with keys pre-filled
# ✓ Opens VS Code automatically

# 3. Copy keys to .env (optional — already in shell profile from step 2)
cp .env.example .env

# 4. Run (two terminals)
node server/index.js   # terminal 1 — FADP-gated API server
node agent/index.js    # terminal 2 — interactive paying agent
```

> **Tip:** Always use `npx @fluidwallet/fadp-cli@latest` to skip the npx cache and get the newest version.

## Endpoints (all FADP-gated)

| Endpoint | Cost | Description |
|----------|------|-------------|
| `GET /health` | Free | Server status + pricing |
| `GET /api/price/:coinId` | 0.001 USDC | Live price |
| `GET /api/prices?ids=eth,btc` | 0.005 USDC | Bulk prices |
| `GET /api/trending` | 0.001 USDC | Trending coins |
| `GET /api/markets` | 0.002 USDC | Top 20 by market cap |
| `POST /api/swap-quote` | 0.002 USDC | Swap quote via Fluid SOR |
| `GET /api/identity/:handle` | 0.001 USDC | Resolve Fluid ID → address |
| `GET /api/balance` | 0.001 USDC | Wallet balance |

## Keys

| Key | Where to get | Used for |
|-----|-------------|----------|
| `FLUID_AGENT_KEY` (`fwag_...`) | Auto-generated by CLI | Real on-chain payments |
| `FLDP_API_KEY_NAME` + private key | Auto-generated by CLI | FADP identity / EC signing |

**No keys?** The demo runs in mock mode — no real USDC needed.

## Agent skills

After running the CLI, `FLUID_AGENT_KEY` is exported to `~/.zshrc`. Codex, Cursor, Cline, and any other AI coding agent picks it up automatically in new terminal sessions — no manual `.env` setup needed.
